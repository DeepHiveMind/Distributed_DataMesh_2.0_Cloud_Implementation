**[Website](https://aws.amazon.com/cli/)** | **[Documentation](https://docs.aws.amazon.com/cli/index.html)** | **[GitHub](https://github.com/aws/aws-cli)**

## How to enable?

```
platys init --enable-services AWSCLI
platys gen
```
